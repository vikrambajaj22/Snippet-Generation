snippet_generation.py is the main code.

Before running it, please make sure to install the Python packages mentioned in the report (main: serpscrap, gensim, nltk, sumy, pyemd)

On Windows, to install pyemd, download whl file from https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyemd

Please also download the Google News pre-trained word2vec model: https://drive.google.com/file/d/0B7XkCwpI5KDYNlNUTTlSS21pQmM/edit?usp=sharing
and place it in the same folder as the .py file.
